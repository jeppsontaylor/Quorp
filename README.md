# Quorp

A high-performance, TUI-first text editor and agentic interface.

Quorp is built on the robust, native Rust foundation of the GPUI research project, stripped of its GUI dependencies to provide a lightning-fast, terminal-native experience with deep integration for AI agents and high-fidelity text rendering.

## Status

Quorp is currently in its early structural "gutting" phase, transitioning from the Zed backend into a standalone, TUI-first project.
