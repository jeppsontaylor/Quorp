# 3. The most important architecture decision

## Do NOT start with classic Ratatui-style boxes everywhere

The issue users often call “pixelated Rust TUI” is usually not Rust itself.

It is usually a combination of:

* terminal emulator quality
* font choice
* font fallback
* DPI / Retina handling
* too much box drawing
* poor redraw strategy
* low-contrast layout
* too much dense screen furniture

## Best rendering model

Build QUORP as a **hybrid terminal renderer**:

### Mode A — transcript mode

Normal terminal scrollback, streaming cards, no “full dashboard” feel.

Use this for:

* reasoning summaries
* tool calls
* shell logs
* diffs
* tests
* plans
* approvals

### Mode B — focused full-screen mode

Alternate-screen only when needed.

Use this for:

* diff review
* file picker
* approval queue
* agent board
* command palette
* settings / theme preview

This gives you the sharp feel of Claude/Codex-style UX without becoming a boxy TUI.

# 4. Best Rust technology stack

Here is the stack I would recommend.

---

## 4.1 Core CLI / app runtime

| Area           | Stack                            | Why                                                |
| -------------- | -------------------------------- | -------------------------------------------------- |
| Language       | **Rust**                         | Performance, portability, strong tooling           |
| Async runtime  | **tokio**                        | model streaming, subprocess management, tool calls |
| CLI parsing    | **clap**                         | mature, ergonomic                                  |
| Config         | **serde + toml + directories**   | typed config and themes                            |
| Logging        | **tracing + tracing-subscriber** | observability and replay                           |
| Error handling | **anyhow + thiserror**           | clean failures                                     |

---

## 4.2 Terminal rendering

This is the critical part.

| Area                       | Stack                                      | Why                                          |
| -------------------------- | ------------------------------------------ | -------------------------------------------- |
| Terminal control           | **crossterm**                              | cross-platform terminal IO, cursor, raw mode |
| Advanced terminal features | **termwiz** (optional)                     | richer terminal protocol support             |
| Direct rendering           | **custom ANSI compositor**                 | avoids the boxed TUI feeling                 |
| Optional layout help       | **ratatui only for some internal widgets** | but not as the primary mental model          |

### Recommendation

Use:

* **Crossterm**
* **direct ANSI / cursor placement**
* **your own scene renderer**
* **minimal Ratatui usage, if any**

So the rendering architecture should be:

```text
App Events
  -> State Reducer
  -> UI Model
  -> Layout Pass
  -> Terminal Frame
  -> Dirty Region Diff
  -> ANSI Output
```

---

## 4.3 Text / Unicode / emoji support

| Area              | Stack                                              | Why                                 |
| ----------------- | -------------------------------------------------- | ----------------------------------- |
| Grapheme handling | **unicode-segmentation**                           | correct cursor movement / slicing   |
| Width calculation | **unicode-width**                                  | correct layout with emoji / symbols |
| Text buffer       | **ropey**                                          | scalable text editing and streaming |
| Emoji support     | terminal font fallback + runtime feature detection | essential for QUORP personality     |

### Important

You specifically want **less-common emoji**. That is fine, but you need:

* width-aware layout
* emoji fallback behavior
* monochrome fallback mode
* terminal capability detection

## `src/main.rs`

```rust
use crossterm::{
    cursor::{Hide, MoveTo, Show},
    execute, queue,
    style::Print,
    terminal::{
        self, disable_raw_mode, enable_raw_mode, Clear, ClearType, EnterAlternateScreen,
        LeaveAlternateScreen,
    },
};
use std::{
    io::{self, stdout, Stdout, Write},
    thread,
    time::Duration,
};
use unicode_width::UnicodeWidthStr;

const RESET: &str = "\x1b[0m";
const BOLD: &str = "\x1b[1m";
const DIM: &str = "\x1b[2m";

#[derive(Copy, Clone)]
struct Rgb(u8, u8, u8);

const BG_DEEP: Rgb = Rgb(5, 5, 10);
const TEXT_PRIMARY: Rgb = Rgb(248, 250, 252);
const TEXT_SECONDARY: Rgb = Rgb(203, 213, 225);
const TEXT_MUTED: Rgb = Rgb(100, 116, 139);

const ORANGE: Rgb = Rgb(255, 122, 26);
const ORANGE_HOT: Rgb = Rgb(255, 155, 61);
const AMBER: Rgb = Rgb(255, 179, 71);
const RED: Rgb = Rgb(255, 77, 90);
const GREEN: Rgb = Rgb(57, 255, 20);
const CYAN: Rgb = Rgb(34, 211, 238);
const BLUE: Rgb = Rgb(96, 165, 250);
const VIOLET: Rgb = Rgb(167, 139, 250);
const MAGENTA: Rgb = Rgb(255, 79, 216);
const BODY: Rgb = Rgb(22, 24, 39);
const BODY_EDGE: Rgb = Rgb(38, 43, 66);
const SHADOW: Rgb = Rgb(11, 16, 32);

const VERSION: &str = "v0.1.0-alpha";

const SPINNER: &[&str] = &["⠋", "⠙", "⠹", "⠸", "⠼", "⠴", "⠦", "⠧", "⠇", "⠏"];

const STEPS: &[(&str, &str, Rgb)] = &[
    ("🪐", "booting agent core", VIOLET),
    ("🧲", "indexing project context", CYAN),
    ("🧰", "wiring toolchain", BLUE),
    ("🧪", "warming test runner", BLUE),
    ("🪄", "priming diff engine", MAGENTA),
    ("🧿", "safety systems online", GREEN),
];

const PROMPT_ART: &[&str] = &[
    "     ▄▄▄▄▄              ",
    "          ▜██           ",
    "             ▜██        ",
    "                ▜██     ",
    "             ▟██▀       ",
    "          ▟██▀          ",
    "     ▀▀▀▀▀              ",
    "                        ",
    "     ▄▄▄▄▄▄▄▄▄▄▄        ",
];

const MASCOT_ART: &[&str] = &[
    "             ╭──╮      ╭──╮           ",
    "          ╭──╯  ╰──────╯  ╰──╮        ",
    "        ╭─╯                    ╰─╮    ",
    "       ╱                          ╲   ",
    "      │                            │  ",
    "      │                            │  ",
    "      │                            │  ",
    "       ╲                          ╱   ",
    "        ╰╮      ╭────────╮     ╭─╯    ",
    "          ╰─────╯        ╰─────╯      ",
    "                                 ╲    ",
    "                                  ╲___",
    "                                     ●─●─●",
];

fn fg(c: Rgb) -> String {
    format!("\x1b[38;2;{};{};{}m", c.0, c.1, c.2)
}

fn bg(c: Rgb) -> String {
    format!("\x1b[48;2;{};{};{}m", c.0, c.1, c.2)
}

fn style(text: &str, color: Rgb) -> String {
    format!("{}{}{}", fg(color), text, RESET)
}

fn style_bold(text: &str, color: Rgb) -> String {
    format!("{}{}{}{}", BOLD, fg(color), text, RESET)
}

fn style_dim(text: &str, color: Rgb) -> String {
    format!("{}{}{}{}", DIM, fg(color), text, RESET)
}

fn lerp(a: u8, b: u8, t: f32) -> u8 {
    (a as f32 + (b as f32 - a as f32) * t).round() as u8
}

fn blend(a: Rgb, b: Rgb, t: f32) -> Rgb {
    Rgb(lerp(a.0, b.0, t), lerp(a.1, b.1, t), lerp(a.2, b.2, t))
}

fn width(s: &str) -> usize {
    UnicodeWidthStr::width(s)
}

fn center_x(cols: u16, content_width: usize) -> u16 {
    if cols as usize > content_width {
        ((cols as usize - content_width) / 2) as u16
    } else {
        0
    }
}

fn write_at(out: &mut Stdout, x: u16, y: u16, s: &str) -> io::Result<()> {
    queue!(out, MoveTo(x, y), Print(s))
}

fn progress_bar(progress: f32, width: usize) -> String {
    let filled = ((progress * width as f32).round() as usize).min(width);
    let mut out = String::new();

    for i in 0..width {
        if i < filled {
            let t = i as f32 / (width.max(1) as f32);
            let c = blend(ORANGE, AMBER, t);
            out.push_str(&format!("{}█{}", fg(c), RESET));
        } else {
            out.push_str(&format!("{}░{}", fg(SHADOW), RESET));
        }
    }

    out
}

fn render_prompt(out: &mut Stdout, x: u16, y: u16, pulse: f32) -> io::Result<()> {
    for (i, line) in PROMPT_ART.iter().enumerate() {
        let t = i as f32 / (PROMPT_ART.len() as f32);
        let hotness = 0.35 + pulse * 0.55;
        let c = blend(ORANGE, ORANGE_HOT, (t * hotness).min(1.0));
        write_at(out, x, y + i as u16, &format!("{}{}{}", fg(c), line, RESET))?;
    }
    Ok(())
}

fn render_mascot(out: &mut Stdout, x: u16, y: u16, pulse: f32) -> io::Result<()> {
    for (i, line) in MASCOT_ART.iter().enumerate() {
        let c = if i < 2 {
            blend(BODY_EDGE, BODY, 0.45)
        } else {
            BODY
        };
        write_at(out, x, y + i as u16, &format!("{}{}{}", fg(c), line, RESET))?;
    }

    let eye = if pulse > 0.5 { ORANGE_HOT } else { AMBER };
    let cheek = blend(ORANGE, AMBER, pulse);
    let tail = blend(AMBER, ORANGE_HOT, pulse);

    // Eyes
    write_at(out, x + 8, y + 4, &format!("{}◉{}", fg(eye), RESET))?;
    write_at(out, x + 27, y + 4, &format!("{}◉{}", fg(eye), RESET))?;

    // Mouth
    write_at(out, x + 16, y + 5, &format!("{}◜ω◝{}", fg(TEXT_PRIMARY), RESET))?;

    // Tiny cheek glow
    write_at(out, x + 30, y + 5, &format!("{}•{}", fg(cheek), RESET))?;

    // Tail glow
    write_at(out, x + 37, y + 12, &format!("{}●─●─●{}", fg(tail), RESET))?;

    Ok(())
}

fn render_header(out: &mut Stdout, cols: u16, y: u16) -> io::Result<()> {
    let title = format!("{}QUORP{}", style_bold("", ORANGE), RESET);
    let line = format!(
        "{}QUORP{}  {}//{}  {}agent shell{}  {}{}{}",
        BOLD,
        RESET,
        style_dim("", TEXT_MUTED),
        RESET,
        style("", TEXT_SECONDARY),
        RESET,
        style_dim("", TEXT_MUTED),
        VERSION,
        RESET,
    );

    let text = format!(
        "{}QUORP{}  {}//{}  {}brilliant terminal IDE{}  {}{}{}",
        BOLD,
        RESET,
        fg(TEXT_MUTED),
        RESET,
        fg(TEXT_SECONDARY),
        RESET,
        fg(TEXT_MUTED),
        VERSION,
        RESET
    );

    let x = center_x(cols, width("QUORP  //  brilliant terminal IDE  v0.1.0-alpha"));
    let _ = title; // keep brand concept explicit
    let _ = line;
    write_at(out, x, y, &text)
}

fn render_footer(out: &mut Stdout, cols: u16, rows: u16) -> io::Result<()> {
    let footer = format!(
        "{}Ghostty{}  {}•{}  {}WezTerm{}  {}•{}  {}Kitty{}  {}recommended for crisp output{}",
        fg(TEXT_MUTED),
        RESET,
        fg(TEXT_MUTED),
        RESET,
        fg(TEXT_MUTED),
        RESET,
        fg(TEXT_MUTED),
        RESET,
        fg(TEXT_MUTED),
        RESET,
        fg(TEXT_MUTED),
        RESET
    );

    let x = center_x(cols, width("Ghostty • WezTerm • Kitty recommended for crisp output"));
    write_at(out, x, rows.saturating_sub(2), &footer)
}

fn render_compact(
    out: &mut Stdout,
    cols: u16,
    rows: u16,
    spinner_idx: usize,
    step_idx: usize,
    pulse: f32,
) -> io::Result<()> {
    let (emoji, label, color) = STEPS[step_idx];
    queue!(out, Clear(ClearType::All))?;

    let title = format!(
        "{}QUORP{}  {}{}{}",
        style_bold("", ORANGE),
        RESET,
        fg(TEXT_MUTED),
        VERSION,
        RESET
    );

    let line1 = format!(
        "{}QUORP{} {}{}{}",
        BOLD,
        RESET,
        fg(TEXT_MUTED),
        VERSION,
        RESET
    );

    let mascot = format!(
        "{}>_{}   {}(◉ᵕ◉)~~~{}",
        fg(blend(ORANGE, ORANGE_HOT, pulse)),
        RESET,
        fg(BODY_EDGE),
        RESET
    );

    let status = format!(
        "{} {}{}{}  {}{}{}",
        SPINNER[spinner_idx % SPINNER.len()],
        fg(color),
        emoji,
        RESET,
        fg(TEXT_PRIMARY),
        label,
        RESET
    );

    let x1 = center_x(cols, width("QUORP v0.1.0-alpha"));
    let x2 = center_x(cols, width(">_   (◉ᵕ◉)~~~"));
    let x3 = center_x(cols, width("⠋ 🪐 booting agent core"));

    write_at(out, x1, rows / 2 - 1, &line1)?;
    let _ = title;
    write_at(out, x2, rows / 2 + 1, &mascot)?;
    write_at(out, x3, rows / 2 + 3, &status)?;
    out.flush()?;
    Ok(())
}

fn render_frame(
    out: &mut Stdout,
    spinner_idx: usize,
    step_idx: usize,
    inner_tick: usize,
) -> io::Result<()> {
    let (cols, rows) = terminal::size().unwrap_or((120, 40));
    let pulse = ((inner_tick as f32 / 18.0) * std::f32::consts::PI).sin().abs();

    if cols < 96 || rows < 28 {
        return render_compact(out, cols, rows, spinner_idx, step_idx, pulse);
    }

    queue!(out, Clear(ClearType::All))?;

    // subtle background wash line
    let top_band = format!("{}{}{}", bg(BG_DEEP), " ".repeat(cols as usize), RESET);
    write_at(out, 0, 0, &top_band)?;

    let hero_width = 88usize;
    let hero_x = center_x(cols, hero_width);
    let hero_y = rows.saturating_sub(24) / 2;

    render_header(out, cols, hero_y.saturating_sub(2))?;

    // left prompt
    let prompt_x = hero_x;
    let prompt_y = hero_y + 4;
    render_prompt(out, prompt_x, prompt_y, pulse)?;

    // right mascot
    let mascot_x = hero_x + 24;
    let mascot_y = hero_y + 1;
    render_mascot(out, mascot_x, mascot_y, pulse)?;

    // Brand line under hero
    let brand = format!(
        "{}QUORP{}  {}•{}  {}sharp terminal IDE{}  {}•{}  {}agentic coding cockpit{}",
        BOLD,
        RESET,
        fg(TEXT_MUTED),
        RESET,
        fg(TEXT_PRIMARY),
        RESET,
        fg(TEXT_MUTED),
        RESET,
        fg(TEXT_SECONDARY),
        RESET
    );
    let brand_x = center_x(cols, width("QUORP • sharp terminal IDE • agentic coding cockpit"));
    write_at(out, brand_x, hero_y + 16, &brand)?;

    // Progress
    let total_steps = STEPS.len() as f32;
    let progress = (step_idx as f32 + (inner_tick as f32 / 18.0)) / total_steps;
    let bar = progress_bar(progress.min(1.0), 38);
    let bar_label = format!(
        "{}loading{}  {}{:>3}%{}",
        fg(TEXT_MUTED),
        RESET,
        fg(TEXT_PRIMARY),
        (progress * 100.0).round() as u8,
        RESET
    );
    let bar_line = format!("{}  {}", bar, bar_label);
    let bar_x = center_x(cols, 38 + 2 + width("loading 100%"));
    write_at(out, bar_x, hero_y + 18, &bar_line)?;

    // Active step
    let (emoji, label, color) = STEPS[step_idx];
    let status = format!(
        "{} {}{}{}  {}{}{}",
        SPINNER[spinner_idx % SPINNER.len()],
        fg(color),
        emoji,
        RESET,
        fg(TEXT_PRIMARY),
        label,
        RESET
    );
    let status_x = center_x(cols, width("⠋ 🪐 booting agent core"));
    write_at(out, status_x, hero_y + 20, &status)?;

    // Capability chips
    let chips = format!(
        "{}🧲 context{}   {}🪄 patch{}   {}🧪 tests{}   {}🪬 safe tools{}",
        fg(CYAN),
        RESET,
        fg(MAGENTA),
        RESET,
        fg(BLUE),
        RESET,
        fg(AMBER),
        RESET
    );
    let chips_x = center_x(cols, width("🧲 context   🪄 patch   🧪 tests   🪬 safe tools"));
    write_at(out, chips_x, hero_y + 22, &chips)?;

    render_footer(out, cols, rows)?;

    out.flush()?;
    Ok(())
}

fn render_ready(out: &mut Stdout) -> io::Result<()> {
    let (cols, rows) = terminal::size().unwrap_or((120, 40));
    queue!(out, Clear(ClearType::All))?;

    let hero = format!(
        "{}QUORP{}  {}ready{}",
        style_bold("", ORANGE),
        RESET,
        fg(GREEN),
        RESET
    );

    let line1 = format!("{}QUORP{}  {}", BOLD, RESET, VERSION);
    let line2 = format!(
        "{}>_{}   {}🫧 ready to ship code{}",
        fg(ORANGE_HOT),
        RESET,
        fg(GREEN),
        RESET
    );
    let line3 = format!(
        "{}type a task or press / for commands{}",
        fg(TEXT_SECONDARY),
        RESET
    );

    let x1 = center_x(cols, width("QUORP v0.1.0-alpha"));
    let x2 = center_x(cols, width(">_ 🫧 ready to ship code"));
    let x3 = center_x(cols, width("type a task or press / for commands"));

    write_at(out, x1, rows / 2 - 1, &line1)?;
    let _ = hero;
    write_at(out, x2, rows / 2 + 1, &line2)?;
    write_at(out, x3, rows / 2 + 3, &line3)?;
    out.flush()?;
    Ok(())
}

fn main() -> io::Result<()> {
    let mut out = stdout();

    enable_raw_mode()?;
    execute!(out, EnterAlternateScreen, Hide)?;

    for step_idx in 0..STEPS.len() {
        for inner_tick in 0..18 {
            let spinner_idx = step_idx * 18 + inner_tick;
            render_frame(&mut out, spinner_idx, step_idx, inner_tick)?;
            thread::sleep(Duration::from_millis(70));
        }
    }

    render_ready(&mut out)?;
    thread::sleep(Duration::from_millis(1100));

    execute!(out, Show, LeaveAlternateScreen)?;
    disable_raw_mode()?;

    println!(
        "{}QUORP{} {}{}{}",
        BOLD,
        RESET,
        fg(TEXT_MUTED),
        VERSION,
        RESET
    );
    println!(
        "{}>_{} {}ready.{}",
        fg(ORANGE_HOT),
        RESET,
        fg(GREEN),
        RESET
    );

    Ok(())
}
```
