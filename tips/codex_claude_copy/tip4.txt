## Executive direction

Build **QUORP** as a **terminal-native agent IDE**, not a traditional full-screen boxy TUI. The winning feel is:

```text
Rust agent core
  + append-only streaming terminal UI
  + ANSI truecolor rendering
  + beautiful diff blocks
  + fast tool-call cards
  + optional full-screen review overlays
  + optional native GPU shell later
```

OpenAI currently describes Codex CLI as a local terminal coding agent that can read, change, and run code, and says it is open source and built in Rust. Anthropic describes Claude Code as an agentic coding tool that reads a codebase, edits files, runs commands, and integrates with dev tools across terminal, IDE, desktop, and browser. So the “sharpness” and “fast colorful behavior” you like is not anti-Rust; it is mostly **streaming terminal design + truecolor + good terminal emulator + careful rendering restraint**. ([OpenAI Developers][1])

# 2. The key rendering decision: not “Rust TUI,” but “terminal stream UI”

A Rust TUI library usually does not render glyph pixels itself. Ratatui, for example, interfaces with the terminal emulator through a backend that draws styled text, manipulates cursor state, and handles terminal properties. The text sharpness ultimately comes from the terminal emulator, font stack, antialiasing, and glyph fallback, not from “Rust vs not Rust.” ([Ratatui][2])

So I would split QUORP into three render modes:

```text
Mode A — default
Streaming ANSI terminal UI.
Normal scrollback. Gorgeous blocks. Fast. Claude/Codex-like.

Mode B — review overlay
Temporary alternate-screen panels for /diff, /review, /agents, /settings.

Mode C — premium native app later
GPUI or Tauri shell using the same Rust agent core.
```

Do **not** start with a permanent full-screen Ratatui dashboard. Start with a high-polish, append-only terminal transcript that feels alive.

# 7. Tool calling and reasoning UX

Claude Code’s docs emphasize that tools make the system agentic: tools let the agent read code, edit files, run commands, search the web, and interact with external services, with each tool result feeding the next decision. ([Claude][6])

QUORP should show a **public reasoning trace**, not hidden chain-of-thought:

```text
⟡ Planning repair path
🜁 Reading parser and failing test
🧬 Found likely newline-preservation mismatch
🪶 Proposing focused patch
🧪 Running focused tests
✦ Verified
```

```text
│  cwd: /repo
│  risk: safe
```

# 9. QUORP load screen design

The load screen should not be a Ratatui dashboard. It should be a **truecolor ANSI splash**.

The trick: render the mascot image as **text** using Unicode half-blocks:

```text
▀
```

Each terminal cell becomes two pixels:

```text
foreground color = upper pixel
background color = lower pixel
character = ▀
```

This produces a surprisingly sharp, colorful, terminal-native mascot without using a GUI, canvas, or image protocol.

Visual structure:

```text
top:
  glowing QUORP wordmark

middle:
  actual mascot converted to ANSI truecolor half-block art

bottom:
  boot card
    version
    worktree guard
    tool forge
    context lattice
    verifier
    approval mode
```

Target boot copy:

```text
QUORP v0.1.0
terminal familiar online
mode: prism-stream
renderer: ansi truecolor
tools: fs / git / shell / lsp / mcp
guard: safe-auto
```

## `src/main.rs`

```rust
use anyhow::{Context, Result};
use image::imageops::FilterType;
use image::RgbaImage;
use std::env;
use std::io::{self, Write};
use std::path::Path;
use std::thread;
use std::time::Duration;
use terminal_size::{terminal_size, Height, Width};

const RESET: &str = "\x1b[0m";
const BOLD: &str = "\x1b[1m";
const DIM: &str = "\x1b[2m";
const HIDE_CURSOR: &str = "\x1b[?25l";
const SHOW_CURSOR: &str = "\x1b[?25h";
const CLEAR: &str = "\x1b[2J";
const HOME: &str = "\x1b[H";

#[derive(Clone, Copy)]
struct Rgb {
    r: u8,
    g: u8,
    b: u8,
}

impl Rgb {
    const fn new(r: u8, g: u8, b: u8) -> Self {
        Self { r, g, b }
    }

    fn fg(self) -> String {
        format!("\x1b[38;2;{};{};{}m", self.r, self.g, self.b)
    }

    fn bg(self) -> String {
        format!("\x1b[48;2;{};{};{}m", self.r, self.g, self.b)
    }
}

#[derive(Clone)]
struct AnsiLine {
    text: String,
    width: usize,
}

impl AnsiLine {
    fn new(text: String, width: usize) -> Self {
        Self { text, width }
    }
}

mod color {
    use super::Rgb;

    pub const VOID: Rgb = Rgb::new(5, 5, 6);
    pub const PANEL: Rgb = Rgb::new(11, 13, 20);
    pub const OBSIDIAN: Rgb = Rgb::new(18, 16, 24);
    pub const SMOKE: Rgb = Rgb::new(47, 43, 54);
    pub const TEXT: Rgb = Rgb::new(244, 247, 255);
    pub const MUTED: Rgb = Rgb::new(104, 112, 134);
    pub const ASH: Rgb = Rgb::new(167, 176, 197);
    pub const EMBER: Rgb = Rgb::new(250, 92, 14);
    pub const EMBER_HOT: Rgb = Rgb::new(255, 122, 18);
    pub const GOLD: Rgb = Rgb::new(254, 191, 54);
    pub const VIOLET: Rgb = Rgb::new(179, 107, 255);
    pub const CYAN: Rgb = Rgb::new(0, 229, 255);
    pub const GREEN: Rgb = Rgb::new(22, 241, 149);
    pub const RED: Rgb = Rgb::new(255, 59, 92);
}

fn main() -> Result<()> {
    let args: Vec<String> = env::args().collect();

    let mascot_path = args
        .get(1)
        .cloned()
        .unwrap_or_else(|| "assets/quorp.png".to_string());

    let version = args
        .get(2)
        .cloned()
        .unwrap_or_else(|| env!("CARGO_PKG_VERSION").to_string());

    let animate = !args.iter().any(|a| a == "--no-animate");

    let (term_w, term_h) = current_terminal_size();

    let mascot = match render_image_as_halfblocks(&mascot_path, term_w, term_h, color::VOID) {
        Ok(lines) => lines,
        Err(_) => fallback_mascot_art(),
    };

    run_splash(&mascot, &version, term_w, term_h, animate)?;

    Ok(())
}

fn current_terminal_size() -> (usize, usize) {
    if let Some((Width(w), Height(h))) = terminal_size() {
        (w as usize, h as usize)
    } else {
        (100, 36)
    }
}

fn run_splash(
    mascot: &[AnsiLine],
    version: &str,
    term_w: usize,
    term_h: usize,
    animate: bool,
) -> Result<()> {
    let frames = if animate { 34 } else { 1 };
    let mut stdout = io::stdout();

    write!(stdout, "{HIDE_CURSOR}")?;
    stdout.flush()?;

    for frame in 0..frames {
        let progress = if animate {
            frame as f32 / (frames - 1) as f32
        } else {
            1.0
        };

        let screen = build_screen(mascot, version, term_w, term_h, frame, progress);
        write!(stdout, "{screen}")?;
        stdout.flush()?;

        if animate {
            thread::sleep(Duration::from_millis(48));
        }
    }

    write!(stdout, "{SHOW_CURSOR}")?;
    stdout.flush()?;

    Ok(())
}

fn build_screen(
    mascot: &[AnsiLine],
    version: &str,
    term_w: usize,
    term_h: usize,
    frame: usize,
    progress: f32,
) -> String {
    let mut lines: Vec<AnsiLine> = Vec::new();

    lines.extend(quorp_logo());
    lines.push(center_text_line(
        "terminal familiar for luminous agentic coding",
        color::ASH,
    ));
    lines.push(AnsiLine::new(String::new(), 0));

    lines.extend_from_slice(mascot);

    lines.push(AnsiLine::new(String::new(), 0));
    lines.extend(boot_panel(version, frame, progress, term_w));

    let content_h = lines.len();
    let top_pad = term_h.saturating_sub(content_h) / 2;

    let mut out = String::new();
    out.push_str(CLEAR);
    out.push_str(HOME);
    out.push_str(&color::VOID.bg());

    for _ in 0..top_pad {
        out.push('\n');
    }

    for line in lines {
        out.push_str(&centered(&line, term_w));
        out.push('\n');
    }

    out.push_str(RESET);
    out
}

fn quorp_logo() -> Vec<AnsiLine> {
    let raw = [
        " ██████╗ ██╗   ██╗ ██████╗ ██████╗ ██████╗ ",
        "██╔═══██╗██║   ██║██╔═══██╗██╔══██╗██╔══██╗",
        "██║   ██║██║   ██║██║   ██║██████╔╝██████╔╝",
        "██║▄▄ ██║██║   ██║██║   ██║██╔══██╗██╔═══╝ ",
        "╚██████╔╝╚██████╔╝╚██████╔╝██║  ██║██║     ",
        " ╚══▀▀═╝  ╚═════╝  ╚═════╝ ╚═╝  ╚═╝╚═╝     ",
    ];

    raw.iter()
        .map(|line| gradient_line(line, color::EMBER_HOT, color::GOLD))
        .collect()
}

fn boot_panel(version: &str, frame: usize, progress: f32, term_w: usize) -> Vec<AnsiLine> {
    let panel_w = term_w.saturating_sub(8).clamp(56, 86);
    let spinner = ["◐", "◓", "◑", "◒"][frame % 4];

    let bar_w = panel_w.saturating_sub(26);
    let filled = ((bar_w as f32) * progress).round() as usize;
    let empty = bar_w.saturating_sub(filled);

    let mut bar = String::new();
    bar.push_str(&color::EMBER_HOT.fg());
    bar.push_str(&"█".repeat(filled));
    bar.push_str(&color::SMOKE.fg());
    bar.push_str(&"░".repeat(empty));
    bar.push_str(RESET);

    let title = format!(
        "{}{}{} QUORP v{} {}terminal familiar online{}",
        color::GOLD.fg(),
        spinner,
        RESET,
        version,
        color::ASH.fg(),
        RESET
    );

    let mode = format!(
        "{}mode{} prism-stream   {}renderer{} ansi truecolor   {}guard{} safe-auto",
        color::MUTED.fg(),
        RESET,
        color::MUTED.fg(),
        RESET,
        color::MUTED.fg(),
        RESET
    );

    let tools = format!(
        "{}tools{} fs / git / shell / lsp / mcp   {}trace{} public, compact, auditable",
        color::MUTED.fg(),
        RESET,
        color::MUTED.fg(),
        RESET
    );

    let mut result = Vec::new();
    result.push(panel_line("╭", "╮", panel_w, color::EMBER));
    result.push(panel_content(&title, panel_w));
    result.push(panel_content(&mode, panel_w));
    result.push(panel_content(&tools, panel_w));

    let progress_text = format!(
        "{}boot{} [{}] {:>3}%",
        color::MUTED.fg(),
        RESET,
        bar,
        (progress * 100.0).round() as usize
    );
    result.push(panel_content(&progress_text, panel_w));
    result.push(panel_line("╰", "╯", panel_w, color::EMBER));

    result
}

fn panel_line(left: &str, right: &str, width: usize, color: Rgb) -> AnsiLine {
    let inner = width.saturating_sub(2);
    let raw = format!("{left}{}{right}", "─".repeat(inner));
    let text = format!("{}{}{}", color.fg(), raw, RESET);
    AnsiLine::new(text, width)
}

fn panel_content(content: &str, width: usize) -> AnsiLine {
    let raw_prefix = "│ ";
    let raw_suffix = " │";
    let visible_hint = strip_ansi_width_cheap(content);
    let inner_width = width.saturating_sub(4);
    let pad = inner_width.saturating_sub(visible_hint);

    let text = format!(
        "{}{}{}{}{}{}",
        color::EMBER.fg(),
        raw_prefix,
        RESET,
        content,
        " ".repeat(pad),
        format!("{}{}{}", color::EMBER.fg(), raw_suffix, RESET)
    );

    AnsiLine::new(text, width)
}

fn render_image_as_halfblocks<P: AsRef<Path>>(
    path: P,
    term_w: usize,
    term_h: usize,
    bg: Rgb,
) -> Result<Vec<AnsiLine>> {
    let img = image::open(&path)
        .with_context(|| format!("failed to open mascot image: {}", path.as_ref().display()))?
        .to_rgba8();

    let (x, y, w, h) = trim_bounds(&img);
    let cropped = image::imageops::crop_imm(&img, x, y, w, h).to_image();

    let max_width_cells = if term_w >= 120 {
        70
    } else if term_w >= 96 {
        58
    } else if term_w >= 76 {
        46
    } else {
        term_w.saturating_sub(8).max(30)
    };

    let reserved_rows = 16usize;
    let max_rows = term_h.saturating_sub(reserved_rows).clamp(12, 34);

    let scale_by_width = max_width_cells as f32 / w as f32;
    let scale_by_height = (max_rows as f32 * 2.0) / h as f32;
    let scale = scale_by_width.min(scale_by_height).max(0.02);

    let target_w = ((w as f32 * scale).round() as u32).max(24);
    let target_h = ((h as f32 * scale).round() as u32).max(12);

    let resized: RgbaImage =
        image::imageops::resize(&cropped, target_w, target_h, FilterType::Lanczos3);

    let mut lines = Vec::new();

    let mut y = 0;
    while y < resized.height() {
        let mut line = String::new();

        for x in 0..resized.width() {
            let upper = blend_rgba(resized.get_pixel(x, y), bg);
            let lower = if y + 1 < resized.height() {
                blend_rgba(resized.get_pixel(x, y + 1), bg)
            } else {
                bg
            };

            line.push_str(&upper.fg());
            line.push_str(&lower.bg());
            line.push('▀');
        }

        line.push_str(RESET);

        lines.push(AnsiLine::new(line, resized.width() as usize));
        y += 2;
    }

    Ok(lines)
}

fn trim_bounds(img: &RgbaImage) -> (u32, u32, u32, u32) {
    let width = img.width();
    let height = img.height();

    let mut min_x = width;
    let mut min_y = height;
    let mut max_x = 0;
    let mut max_y = 0;
    let mut found = false;

    for y in 0..height {
        for x in 0..width {
            let p = img.get_pixel(x, y);
            let a = p[3];
            let max_rgb = p[0].max(p[1]).max(p[2]);

            // Keeps the black body, orange glow, and tail bulbs while dropping transparent margins.
            if a > 80 && max_rgb > 8 {
                found = true;
                min_x = min_x.min(x);
                min_y = min_y.min(y);
                max_x = max_x.max(x);
                max_y = max_y.max(y);
            }
        }
    }

    if !found {
        return (0, 0, width, height);
    }

    let pad = 32u32;
    let x = min_x.saturating_sub(pad);
    let y = min_y.saturating_sub(pad);
    let right = (max_x + pad).min(width.saturating_sub(1));
    let bottom = (max_y + pad).min(height.saturating_sub(1));

    (x, y, right.saturating_sub(x) + 1, bottom.saturating_sub(y) + 1)
}

fn blend_rgba(px: &image::Rgba<u8>, bg: Rgb) -> Rgb {
    let a = px[3] as f32 / 255.0;
    let inv = 1.0 - a;

    Rgb {
        r: ((px[0] as f32 * a) + (bg.r as f32 * inv)).round() as u8,
        g: ((px[1] as f32 * a) + (bg.g as f32 * inv)).round() as u8,
        b: ((px[2] as f32 * a) + (bg.b as f32 * inv)).round() as u8,
    }
}

fn fallback_mascot_art() -> Vec<AnsiLine> {
    let raw = [
        "          ▟████▙        ▟████▙          ",
        "        ▟████████▙    ▟████████▙        ",

    raw.iter()
        .map(|line| gradient_line(line, color::SMOKE, color::EMBER_HOT))

}

fn gradient_line(raw: &str, left: Rgb, right: Rgb) -> AnsiLine {
    let chars: Vec<char> = raw.chars().collect();
    let n = chars.len().max(1);

    let mut out = String::new();

    for (i, ch) in chars.iter().enumerate() {
        let t = if n <= 1 {
            0.0
        } else {
            i as f32 / (n - 1) as f32
        };

        let c = mix(left, right, t);
        out.push_str(&c.fg());
        out.push(*ch);
    }

    out.push_str(RESET);

    AnsiLine::new(out, chars.len())
}

fn center_text_line(raw: &str, color: Rgb) -> AnsiLine {
    let text = format!("{}{}{}{}", DIM, color.fg(), raw, RESET);
    AnsiLine::new(text, raw.chars().count())
}

fn centered(line: &AnsiLine, term_w: usize) -> String {
    let pad = term_w.saturating_sub(line.width) / 2;
    format!("{}{}", " ".repeat(pad), line.text)
}

fn mix(a: Rgb, b: Rgb, t: f32) -> Rgb {
    Rgb {
        r: lerp(a.r, b.r, t),
        g: lerp(a.g, b.g, t),
        b: lerp(a.b, b.b, t),
    }
}

fn lerp(a: u8, b: u8, t: f32) -> u8 {
    let a = a as f32;
    let b = b as f32;
    (a + (b - a) * t).round().clamp(0.0, 255.0) as u8
}

// Cheap width estimate that ignores ANSI escape sequences.
// Good enough for the splash card; use unicode-width in production.
fn strip_ansi_width_cheap(s: &str) -> usize {
    let mut width = 0usize;
    let mut chars = s.chars().peekable();

    while let Some(ch) = chars.next() {
        if ch == '\x1b' {
            for c in chars.by_ref() {
                if c == 'm' {
                    break;
                }
            }
        } else {
            width += 1;
        }
    }

    width
}
```
