# Upgrade V2 Notes

This directory contains the recovered v2 advisory notes for Quorp's agent OS roadmap.
They are kept here as raw source notes so the repo can reference them directly instead of
depending on dangling blobs or out-of-band recovery.

The notes cover the main themes called out in the roadmap:

- Repo localization and audit recovery
- Precise edit semantics and patch safety
- Task runtime and controller policy
- Verification, proof, and replay
- Memory, rules, and learning
- Permissions, sandboxing, and MCP
- Worktree isolation and release hardening
